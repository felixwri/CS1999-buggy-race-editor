Features

Guest accounts, so most pages and features are public

Password protected accounts so buggies can be saved securely

All passwords can be changed

Custom themes which are site wide and can be as unique as the user desires

Multiple buggies can be saved, updated or deleted

Auto build can create buggies with a single click

Task 2

2-EDIT

I initially loaded the current values in the placeholder attribute so that the previous value could be seen but didn't have to be removed. However, i decided to use the value attribute instead as it made getting the data for the form alot easier.

2-FORM

Added labels for every input in the form to make it easier to understand

2-COST

I created a javascript object with all the prices for each buggy attribute. I then calculate the cost using a for loop through every input.name and input.value with some edge cases hard coded.

Created a new python file called validation.py which takes a dictionary of all the fields and checks whether they are valid.
